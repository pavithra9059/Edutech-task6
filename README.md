Project Overview

This project demonstrates how to consume REST APIs using Axios by interacting with the mock API service provided by JSONPlaceholder.
The application performs GET and POST requests while managing loading states and rendering dynamic, data-driven UI components.

🚀 Features
-> Fetch data from REST API (GET request)
-> Create new data entries (POST request)
-> Loading state handling for better UX
-> Dynamic rendering of API data
-> Error handling for failed requests
-> Clean and modular API integration

* Tech Stack
->JavaScript (ES6+)
->HTML5 / CSS3
->Axios
->REST API (JSONPlaceholder)
